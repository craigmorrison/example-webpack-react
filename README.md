# Basic Webpack Example

An example project for teaching build configuration with React.

## Commands

## Install

To install use:

```shell
npm install
```

## Test

To run the test suite use:

```
npm test
```

## Run

To run the site use:

```shell
npm start
```

## Build

To build a deployable version use:

```shell
npm run build
```

## Lint

To check the code style with ESLint run:

```shell
npm run lint
```
